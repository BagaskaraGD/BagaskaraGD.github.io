Commands & concepts I keep referring back to

Browse all notes →
Laravel #001

Run database migrations

$php artisan migrate --seed

Applies pending migrations and runs seeders right after. Use on fresh setup or to refresh a dev environment quickly.

3 min read Open →
Git #014

Undo the last commit (keep changes)

$git reset --soft HEAD~1

Moves HEAD back one commit but leaves files staged. Perfect when you committed too early and want to rework the message or split the change.

2 min read Open →
Docker #022

Stop all running containers

$docker stop $(docker ps -q)

Quickly halt every running container. Handy after a long dev session before switching projects or rebooting the machine.

1 min read Open →
Flutter #030

Clean & rebuild after a weird crash

$flutter clean && flutter pub get

When the build cache lies to you. Wipes build/ and re-resolves dependencies — fixes the majority of "it worked yesterday" issues.

2 min read Open →
MySQL #041

Dump a single table for backup

$mysqldump -u root -p db users > users.sql

Exports just one table — fast and small. Useful for sharing a sample dataset or restoring specific data without touching the rest.

2 min read Open →
Terminal #055

Find files modified in the last hour

$find . -mmin -60 -type f

Lists files changed in the last 60 minutes. Great for tracking down what got touched during a debugging spree.
